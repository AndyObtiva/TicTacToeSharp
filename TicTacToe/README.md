# Tic Tac Toe

## Requirements

* Player 1 plays symbol “X”, player 2 plays symbol “O”.
* Use the numpad for input (accept numbers 1 through 9):
  7 | 8 | 9
  4 | 5 | 6
  1 | 2 | 3
* Indicate the outcome of the game: If player 1 wins, print “Player X Won!” – If player 2 wins, print “Player O Won!” – Otherwise print “Cat’s Game!”

## Use Cases

* Make Move
* See Board State (including win/lose or draw)

## Algorithm Specification

* Start game with player X
* Accept numeric placement locations of X/O until win/lose or draw is achieved
* Identify Win state (3 cases of horizontal win, 3 cases of vertical win, 2 cases of diagonal win)
* Identify Draw state
